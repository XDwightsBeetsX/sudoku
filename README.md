# sudoku

Solve sudoku puzzles in C++

## Usage

1. Download the repo

    ```shell
    > git clone https://github.com/XDwightsBeetsX/sudoku
    ```

2. Make the executable

    ```shell
    > make all
    ```

3. Edit the `input/input.csv` file with the `.csv`-formatted sudoku puzzle

4. Run the solver

    ```shell
    make run
    ```
